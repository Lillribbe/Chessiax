# Chessiax

Chessiax is an AI-powered chess opening trainer that helps players discover, study, and practice chess openings based on rating level and learning goals.

## Features

- Opening recommendations by ELO level
- Interactive opening study board
- Practice mode with move-by-move training
- Mistake review
- Opening library with filters
- Chess.com profile analysis
- AI-generated summaries and recommendations

## Tech Stack

- Python
- Flask
- JavaScript
- HTML/CSS
- Chessboard.js
- Chess.js
- OpenAI API
- Render deployment

## Status

Live MVP launched at: https://chessiax.com

The product is under active development. The next major step is personalized Chess.com game analysis to identify a player’s strongest and weakest openings.
